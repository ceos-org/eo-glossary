# Algorithm

## 1 Definition

Series of steps or rules for solving a problem or performing a task, especially by a computer.

### Notes

Algorithms implemented as software are delivered to science processing system to use as primary tools in the generation of science products. The term includes executable code, source code, job control scripts, and documentation.
