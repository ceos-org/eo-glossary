# Data Use Metrics

## 1 Definition

Data Use Metrics represent statistics on data access volumes, user groups, usefulness/popularity of tools and services.

### Notes

These metrics are used by data managers to report and monitor the usefulness of datasets and products and are an effective means to engage users and stakeholders to improve products and services.
