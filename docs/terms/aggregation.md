# Aggregation

## 1 Definition

The presentation of many data files as one consistent block of time and space.

### Notes

Usually implemented w/ OPENDAP clients THREDDS, ERDDAP Hyrax. Aggregation can be achieved through physical aggregation (combining several netcdf files to present a consistent block of time/space) or through virtual aggregation (combining several netcdf files dynamically through an OPENDAB client (ZarrDAP, THREDDS)
