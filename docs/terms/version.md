# Version

## 1 Definition

Versioning refers to saving new copies of your files when you make changes so that you can go back and retrieve specific versions of your files later.

### Notes

Best practices is to record a version in the metadata. The version is increased when there is a modification to the scientific algorithm used to process the data.
