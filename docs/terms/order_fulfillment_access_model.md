# Order Fulfillment Access Model

## 1 Definition

Group of data access services enabling delayed delivery due to non-immediate availability or need for formatting, transformation, or packaging.

### Notes

These services include:
- Subsetting service - selecting a subset of observations from a dataset for particular time(s), location(s), observation station(s), and output format(s)
- Document certification
- Subscription service - a service allowing users to subscribe to notifications for datasets.
- Ad Hoc orders
- Event based subsetting
- Transformation service
- Bulk ordering - “ could involve pulling data from glacier” - volume component
