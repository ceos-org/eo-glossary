# Reference Model

## 1 Definition

Conceptual framework of unifying concepts for understanding relationships among entities in an environment and developing consistent standards, particularly for archival systems preserving digital information.

### Notes

OAIS Definition Reference Model: a framework for understanding significant relationships among the entities of some environment, and for the development of consistent standards or specifications supporting that environment. A reference model is based on a small number of unifying concepts and may be used as a basis for education and explaining standards to a non-specialist.
