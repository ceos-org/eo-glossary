# Archive Recommendation Package

**Synonyms**: ARP

## 1 Definition

An ARP is written in collaboration with the Data Producers early on in the process, in which recommendations are assembled for what exactly should be archived. They are used for all the satellite missions for example, to support readiness reviews and make recommendations on things like what level of data should be archived.

### Notes

An ARP defines what should be archived to ensure the data are independently understandable. The ARP may include additional data and information required for completeness such as ancillary and auxiliary information, change memos, etc.
