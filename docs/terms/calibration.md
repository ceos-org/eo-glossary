# Calibration

## 1 Definition

Process of quantitatively defining a system's (e.g. a Sensor or Model's) response to known, controlled inputs produced for example by references. In Remote Sensing, it implies the collection of pre-flight and in-flight calibration measurements and in-situ Data that is to be used in the Data calibration processing routine.
