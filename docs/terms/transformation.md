# Transformation

## 1 Definition

A digital migration in which there is an alteration to the content information or preservation Description information of an archival information package. For example, changing ASCII codes to UNICODE in a text document being preserved is a transformation. See also reformatting .

### Notes

Data can be converted dynamically on-demand (eg. converting Grib to netCDF, converting neCDF to COG) using a lambda function. The may result from order fulfillment in converting for example netCDF to Zarr or CSV to PDF.
