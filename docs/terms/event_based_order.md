# Event-based Order

## 1 Definition

An event is any environmental observation of significance.

### Notes

There are two use cases for event based orders. 1) Observations from a particular event (time, space, variable) 2) Using an event observed in one dataset to order/subset data from another dataset. Might require an order processor or AI
