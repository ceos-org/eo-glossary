# Self-Service Access Model

## 1 Definition

Self-Service Access Model - a group of data access services which a user can employ to obtain data of interest without any delay as the data are immediately available. Synchronous requests.

### Notes

These services include
- Subsetting service available through tools (Thredds, Hyrax, ERDDAP)
- Family of transfer services (SFTP, FTP, FTPS, HTTPS)
- Visualization service (Hyrax, Jupyter Notebooks)
- Preview service (Common Access
- Geographic Information Services (GIS) clip-n-ship
- Transformation services (Lambda functions) Convert Grib to ZARR Convert netCDF to COG
- Dependent on the dataset and characteristics
