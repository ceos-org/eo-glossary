# Maturity Matrix

**Synonyms**: Maturity Model

## 1 Definition

Maturity Matrix/Model is a measurement of the ability of an organization for continuous improvement in a particular discipline. It defines of all activities needed to preserve and improve the information content, quality, accessibility, and usability of data and metadata.
